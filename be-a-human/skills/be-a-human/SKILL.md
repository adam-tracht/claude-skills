# Deslop

Make prose sound like a specific human wrote it, not a language model.

## Voice Calibration (optional)

If the user provides a writing sample, analyze it first: sentence length patterns, word choice register, how paragraphs start and transition, punctuation habits. Match those patterns in the output. Without a sample: varied rhythm, opinionated, direct, trusts the reader.

## Core Rules

**1. Cut filler.** No throat-clearing openers ("Here's the thing:"), emphasis crutches ("Let that sink in."), meta-commentary ("In this section, we'll explore..."), or chatbot artifacts ("Great question!", "I hope this helps!").

**2. Break formulaic structures.** No binary contrasts ("Not X. Y."), negative listings, dramatic fragmentation ("Speed. That's it."), or rhetorical questions answered immediately.

**3. Eliminate AI vocabulary.** Remove: *delve, tapestry, landscape, pivotal, crucial, testament, underscore, highlight, showcase, foster, garner, vibrant, nuanced, interplay, intricate, enduring, navigate, ecosystem*. Cut copula avoidance ("serves as," "stands as," "marks a") — use "is" or "has." See [references/patterns.md](references/patterns.md).

**4. Name the actor.** Make the human doing the thing the subject. Passive voice and subjectless fragments ("No configuration needed") hide actors.

**5. Be specific.** Name the implication, not "the implications are significant." Name the expert, not "experts argue." No loose absolutes.

**6. Match register.** Blog/newsletter: "you" beats "people," scenarios beat abstractions. Scientific writing: "we" for your work, cite specific authors, avoid distant narrator voice ("It has long been recognized...").

**7. Vary rhythm.** Mix sentence lengths. No stacked short fragments for manufactured emphasis. No em dashes.

**8. Trust readers.** No "Let's break this down." No "Think of it as..." No fractal summaries (announcing what you'll say, saying it, then summarizing it).

**9. Watch formatting tells.** No bold-first bullets, decorative emojis, em dashes, "In conclusion..." / "Despite these challenges..." formulas, title-case headings, or inline-header bullet lists.

**10. Don't dilute.** One point per section.

## Add Soul

Voiceless writing that's technically clean is still obviously not human: every sentence the same length, no opinions, no uncertainty, reads like a press release. Have opinions, react to facts, vary rhythm, use "I" when it fits, be specific about feelings ("there's something unsettling about agents churning away at 3am" beats "this is concerning").

## Quick Checks

- Heavy adverbs? Cut.
- Passive voice? Find the actor.
- Inanimate thing doing a human verb? Name the person.
- Throat-clearing opener? Cut to the point.
- "Not X, it's Y"? State Y directly.
- Rhetorical question answered immediately? Fold into a statement.
- Three consecutive sentences match length? Break one.
- Em dash? Replace (comma, period, or parenthetical).
- Vague declarative? Name the specific thing.
- "It's worth noting" / meta-joiners? Delete.
- Same metaphor more than twice? Cut repeats.
- "Despite these challenges..."? Rewrite.
- Bold-first bullet pattern? Remove bold leads.
- Tricolon? Use two items or one.
- AI vocabulary (delve, tapestry, pivotal...)? Replace.
- "Serves as" / "stands as"? Replace with "is."
- Sycophantic opener or closer? Cut.

## Process

1. If a voice sample was provided, analyze it first.
2. Edit using the core rules; consult [references/patterns.md](references/patterns.md) for the full pattern catalog.
3. **Self-audit:** Ask internally: "What makes this still obviously AI-generated?" List the remaining tells. Fix them.
4. Deliver the final version.

If reviewing rather than editing: score first, then give targeted feedback with specific line-level examples.

Rate 1–10 on each:
